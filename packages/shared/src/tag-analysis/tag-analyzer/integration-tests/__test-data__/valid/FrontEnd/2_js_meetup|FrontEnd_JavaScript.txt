JavaScript meetup #1

12 сентября в офисе компании DINS состоится открытый JavaScript митап. Поговорим про абстрактные синтаксические деревья, Protobuf и тестирование web UI с углублением в незначительные(значительные) детали.
1. “Абстрактные синтаксические деревья в JavaScript”, Евгения Ольнева

В докладе мы рассмотрим, что это за зверь такой «абстрактное синтаксическое дерево». Попробуем понять, как оно используется в парсинге JavaScript кода, и где еще находит применение в разработке на JavaScript. Также мы решим несколько повседневных задач с его помощью.

2. “Тестирование web UI с углублением в незначительные детали”, Сергей Журавлев

Идея о том, что тестирование необходимо как класс, уже давно вошла в жизнь фронтенд разработчиков, но большинство примеров, которые можно найти, либо слишком упрощенные, либо немасштабируемые, либо просто непопулярные. Попробуем разобрать, что к чему в тестировании UI приложений, а также выясним, какие подходы могут оказаться лучшим решением.

3. “Protobuf: как быстро начать использовать. Его плюсы и минусы”, Андрей Советкин

Protobuf - это протокол межсетевого общения, который позволяет общаться с бекендом на одном языке. В докладе мы посмотрим, как же быстро начать его использовать, а также разберем его основные плюсы и минусы.
