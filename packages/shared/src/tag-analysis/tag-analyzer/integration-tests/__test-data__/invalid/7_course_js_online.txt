Приглашаем на второй блок курса по веб-разработке
Этот курс проводится в рамках курса по web-разработке и является его вторым блоком. Первый блок посвящен изучению базовых основ HTML и CSS (верстки сайта), необходимых для полноценного овладения front-end-разработкой. Для прохождения курса потребуется английский на уровне чтения документации.

 

Благодаря курсу Вы:

 


	Получите базовые знания языка JavaScript и сможете работать по новой профессии. Выпускник становится Junior JavaScript Developer или Front-end Developer.

 


	Сможете самостоятельно создавать известные игры - “Охоту на уток”, “Футбол” и “Мортал Комбат”. Научитесь писать свои первые приложения, которые и войдут в Ваше портфолио.

 


	Научитесь оформлять резюме и портфолио в соответствии с запросами компаний, зарегистрируетесь и заполните свои аккаунты на профессиональных платформах, подготовитесь к прохождению собеседований. Для всего этого мы специально ввели новый блок - по трудоустройству выпускников.

 


	Получите отличные перспективы для дальнейшего развития  в качестве JavaScript-разработчика, возможность работать в смежных направлениях, сильную базу для освоения других языков программирования.


 

Специфика курса: для начинающих. Упор делается на практические занятия, в том числе, на изучение JQuery.

 

Длительность курса:  40 часов

 

Место проведения: занятия проходят в очном формате в центре Новосибирска или Академгородке и в формате онлайн для жителей любого города или страны.


Программа:

I блок

1) Переменные и типы данных.
2) Условные операторы и операторы сравнения.
3) Циклы.
4) Функции, рекурсивные функции, замыкание.
5) Объекты, свойства, методы, this.
6) Массивы.
7) Работа с датой и временем.
8) Таймеры.
9) Пользовательские события.
10) Регулярные выражения.
11) Что такое DOM и BOM? Работа с DOM. 
12) Объекты Location, History, Screen, Navigator. Работа с ними.
13) Правильное оформление кода.
14) Подключаем JQUERY.
15) Работаем с селекторами, обработчиками событий, изменяем страницу.
16) AJAX.17) JQuery UI.18) Сборщик проектов WEBPACK.19) GIT.

II блок - трудоустройство после курса (3 часа) - бесплатно

1) Оформление резюме и портфолио

2) Основные платформы: регистрация и заполнения аккаунтов

3) Практическая работа

4) Подготовка к собеседованию, обзор основных вопросов и кейсов
