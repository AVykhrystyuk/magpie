[event_id: 802705] Курс "Современная web-разработка" для школьников (12-17 лет)

Занятия по воскресеньям с 10.00 до 14.00.
Возможна оплата в рассрочку.
Курс «Современная web-разработка»   предназначен для слушателей, которые хотят научиться разрабатывать сайты, начиная с блока представления (html, css, js) и заканчивая логикой (php).

 

Курс включает в себя:

Знакомство с технологиями вёрстки (html, css, javascript)
	Обучение работе с каскадными таблицами стилей
	Обучение работе с библиотеками (bootstrap, др)
	Знакомство с клиент-серверными приложениями
	Обучение работе с PHP.
 

Планируемые результаты обучения

По окончании обучения слушатель:

обладает практическими навыками создания сайтов;
	ориентируется в базовых технологиях, используемых в современной веб-разработке, таких, как: HTML, CSS, JavaScript, jQuery, PHP и т. д.\
